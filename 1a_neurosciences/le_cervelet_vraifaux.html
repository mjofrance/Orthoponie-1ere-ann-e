QCM : Le Cervelet (Anatomie et Physiologie)

1. Le cervelet est relié au tronc cérébral par les pédoncules cérébraux.

Vrai
Faux
Réponse : FAUX

Le cervelet est relié au tronc cérébral grâce à des pédoncules cérébelleux.

2. Le cervelet est constitué de 3 lobes : antérieur, postérieur et flocculo-nodulaire.

Vrai
Faux
Réponse : VRAI

Ceci est vrai pour chacun des deux hémisphères qui composent le cervelet.

3. Chaque lobe a une fonction spécifique dans la motricité.

Vrai
Faux
Réponse : VRAI

4. Le maintien de l'équilibre est assuré par le lobe postérieur.

Vrai
Faux
Réponse : FAUX

C'est le lobe flocculo-nodulaire qui entre en jeu pour le maintien de l'équilibre.

5. Le lobe postérieur intervient pour ajuster vitesse, trajectoire et puissance des mouvements en temps réel.

Vrai
Faux
Réponse : VRAI

6. Le cortex cérébelleux est composé de 6 couches de neurones comme le cortex cérébral.

Vrai
Faux
Réponse : FAUX

Le cortex cérébelleux ne possède que 3 couches (moléculaire, Purkinje, granulaire).

7. Dans le cervelet, la substance grise est périphérique comme dans le cerveau.

Vrai
Faux
Réponse : VRAI

8. Les cellules de Purkinje constituent la couche la plus externe du cortex.

Vrai
Faux
Réponse : FAUX

Elles constituent la couche intermédiaire. La couche la plus externe est la couche moléculaire.

9. La couche granulaire est la couche interne du cortex cérébelleux.

Vrai
Faux
Réponse : VRAI

10. Toutes les afférences du cervelet sont excitatrices.

Vrai
Faux
Réponse : VRAI

11. Les informations afférentes sont véhiculées par les fibres grimpantes et les fibres moussues.

Vrai
Faux
Réponse : VRAI

12. L'information nerveuse sort du cervelet par les cellules granulaires.

Vrai
Faux
Réponse : FAUX

Le message efférent sort par les axones des cellules de Purkinje.

13. Les dendrites des cellules de Purkinje sont dans la couche intermédiaire.

Vrai
Faux
Réponse : FAUX

Leurs dendrites sont déployées dans la couche moléculaire.

14. Les axones des cellules de Purkinje traversent la couche granulaire.

Vrai
Faux
Réponse : VRAI

15. Les fibres moussues sont les axones de neurones situés dans la couche granulaire.

Vrai
Faux
Réponse : FAUX

Les fibres moussues sont des afférences provenant de l'extérieur du cervelet (tronc, moelle).

16. Une fibre parallèle établit de multiples connexions synaptiques avec une seule cellule de Purkinje.

Vrai
Faux
Réponse : VRAI

17. Les cellules de Purkinje sont inhibées grâce à des interneurones.

Vrai
Faux
Réponse : VRAI

18. Le noyau fastigial joue un rôle dans le tonus musculaire.

Vrai
Faux
Réponse : FAUX

Ce sont les noyaux interposés qui jouent un rôle dans le tonus musculaire.

19. Les noyaux cérébelleux sont des amas de substance grise recevant des informations du cortex cérébelleux.

Vrai
Faux
Réponse : VRAI

Ils sont enchâssés dans la substance blanche et communiquent avec différentes régions.

20. L'archéocervelet intervient dans l'équilibre.

Vrai
Faux
Réponse : VRAI

21. Le paléocervelet émet des projections sur le noyau dentelé.

Vrai
Faux
Réponse : FAUX

Le paléocervelet émet des projections sur les noyaux interposés.

22. Le néocervelet communique via les noyaux dentelés puis le thalamus avec le cortex moteur.

Vrai
Faux
Réponse : VRAI

Il reçoit les afférences du cortex cérébral et émet des efférences vers le thalamus et le noyau rouge via le noyau dentelé.
