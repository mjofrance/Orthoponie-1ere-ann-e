QCM : Le Système Nerveux

Exercice : Vrai ou Faux

1. D'un point de vue fonctionnel, le SN peut être subdivisé en SN central et SN périphérique.

Vrai
Faux
Réponse : FAUX

C'est d'un point de vue anatomique.

2. Les ganglions font partie du SN périphérique.

Vrai
Faux
Réponse : VRAI

3. On trouve des méninges dans la colonne vertébrale.

Vrai
Faux
Réponse : VRAI

4. La pie-mère est la méninge accolée à l'os.

Vrai
Faux
Réponse : FAUX

C'est la dure-mère.

5. Le diencéphale est composé du thalamus et de l'hypothalamus.

Vrai
Faux
Réponse : VRAI

(notamment mais pas uniquement)

6. Le liquide céphalorachidien (LCR) circule dans l'espace épidural.

Vrai
Faux
Réponse : FAUX

Il circule dans l'espace sous arachnoïde.

7. Chaque hémisphère cérébral comprend 4 lobes.

Vrai
Faux
Réponse : VRAI

Chaque hémisphère contient bien 4 lobes même s'il existe un 5ème lobe.

8. Le cortex cérébral contient toute la substance grise du cerveau.

Vrai
Faux
Réponse : FAUX

Il y a d'autres substances grises dans les noyaux gris centraux.

9. Le thalamus joue un rôle dans la thermorégulation.

Vrai
Faux
Réponse : FAUX

C'est l'hypothalamus.

10. La glande pinéale est une des structures du diencéphale.

Vrai
Faux
Réponse : VRAI

Elle fait partie de l'épithalamus, elle est également appelée « épiphyse ».

11. Certains tissus nerveux produisent des hormones.

Vrai
Faux
Réponse : VRAI

Exemple : la mélatonine synthétisée par l'épiphyse de l'épithalamus.

12. Le cervelet est le centre de coordination des fonctions autonomes.

Vrai
Faux
Réponse : FAUX

C'est le centre de coordination des mouvements, de l'équilibre et de la posture.

13. Les trois régions du cervelet sont le métencéphale, le pont et le bulbe rachidien.

Vrai
Faux
Réponse : FAUX

Ce sont les régions du tronc cérébral.

14. La moelle épinière s'étend de la première cervicale à la dernière lombaire.

Vrai
Faux
Réponse : FAUX

Elle s'étend jusqu'à la vertèbre lombaire L1.

15. La substance grise est en périphérie dans tout le SNC.

Vrai
Faux
Réponse : FAUX

Dans la moelle épinière, elle est autour du canal de l'épendyme (inverse du cerveau où elle est en périphérie).

16. Les corps cellulaires des neurones sont toujours dans la substance grise.

Vrai
Faux
Réponse : VRAI

Dans la substance grise de la moelle épinière et celle du cerveau.

17. Les corps cellulaires des neurones sont toujours dans le SNC.

Vrai
Faux
Réponse : FAUX

Les corps cellulaires sont toujours dans la substance grise, mais il y en a aussi dans les ganglions (SNP).

18. Les nerfs sensitifs sont efférents.

Vrai
Faux
Réponse : FAUX

Les nerfs sensitifs sont afférents, et les nerfs moteurs sont efférents.

19. Il y a 12 paires de nerfs crâniens et 31 paires de nerfs rachidiens.

Vrai
Faux
Réponse : VRAI

20. La racine postérieure des nerfs rachidiens est motrice.

Vrai
Faux
Réponse : FAUX

La racine postérieure est sensitive.

21. Le rameau dorsal d'un nerf rachidien est sensitif.

Vrai
Faux
Réponse : FAUX

Les rameaux sont mixtes.

22. Un nerf issu d'un plexus contient des fibres de plusieurs nerfs rachidiens.

Vrai
Faux
Réponse : VRAI

23. Les nerfs crâniens innervent uniquement la tête et le cou.

Vrai
Faux
Réponse : FAUX

La Paire X (nerf vague) innerve aussi les régions thoracique et abdominale.

24. Tous les nerfs crâniens sont mixtes.

Vrai
Faux
Réponse : FAUX

Les nerfs olfactifs (I), optiques (II) et vestibulo-cochléaires (VIII) sont uniquement sensitifs.

25. La gaine entourant les nerfs est la périnèvre.

Vrai
Faux
Réponse : FAUX

Le périnèvre entoure le faisceau. La gaine du nerf est l'épinèvre.

26. Un récepteur sensoriel est une cellule spécialisée.

Vrai
Faux
Réponse : VRAI

Mais cela peut aussi être la partie terminale d'un neurone sensoriel.

27. Les récepteurs sensoriels des organes des sens sont des intérocepteurs.

Vrai
Faux
Réponse : FAUX

Ce sont des extérocepteurs.

28. Les récepteurs sensoriels font partie du SNC.

Vrai
Faux
Réponse : FAUX

Ils font partie du SNP.

29. Le cortex cérébral est formé de 6 couches de neurones.

Vrai
Faux
Réponse : VRAI

30. L'épaisseur des couches est identique dans tout le cortex.

Vrai
Faux
Réponse : FAUX

Les 6 couches sont présentes partout, mais leur épaisseur varie en fonction des zones et de leur rôle.

31. Les couches du cortex représentent les unités fonctionnelles de traitement de l'information.

Vrai
Faux
Réponse : FAUX

Ce sont les colonnes (toutes les couches d'une même colonne traitent la même information).

32. Les aires cérébrales sensorielles reçoivent les messages des récepteurs sensoriels.

Vrai
Faux
Réponse : VRAI

33. Les neurones sont les cellules les plus abondantes du tissu nerveux.

Vrai
Faux
Réponse : FAUX

Ce sont les cellules gliales.

34. Les zones de réception des messages sont les dendrites des neurones.

Vrai
Faux
Réponse : VRAI

35. Un neurone est souvent constitué de nombreuses dendrites, d'un corps cellulaire et de plusieurs axones.

Vrai
Faux
Réponse : FAUX

L'axone est toujours unique.

36. Les corps de Nissl sont présents dans tout le neurone.

Vrai
Faux
Réponse : FAUX

Ils ne sont pas présents dans l'axone ni le cône d'émergence.

37. Le transport vers l'extrémité de l'axone est le transport antérograde.

Vrai
Faux
Réponse : VRAI

Il implique les kinésines.

38. Le transport rétrograde utilise la dynéine comme protéine motrice.

Vrai
Faux
Réponse : VRAI

39. Seul l'axone peut être myélinisé.

Vrai
Faux
Réponse : VRAI

Les dendrites ne sont pas myélinisées.

40. La gaine de myéline est formée par des cellules gliales.

Vrai
Faux
Réponse : VRAI

41. La gaine de myéline est continue le long de l'axone.

Vrai
Faux
Réponse : FAUX

Il y a des interruptions : les nœuds de Ranvier.

42. Les cellules de Schwann sont les cellules produisant la gaine de myéline dans le SNC.

Vrai
Faux
Réponse : FAUX

Les cellules de Schwann agissent dans le SNP ; dans le SNC, ce sont les oligodendrocytes.

43. Les astrocytes permettent les échanges entre le sang et les neurones.

Vrai
Faux
Réponse : VRAI

44. La microglie joue un rôle immunitaire dans le cerveau.

Vrai
Faux
Réponse : VRAI

Elle joue un rôle de macrophage.
