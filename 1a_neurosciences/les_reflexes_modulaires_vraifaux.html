QCM : Les réflexes médullaires

Exercice : Vrai ou Faux

1. Un muscle se relâche lorsqu’il reçoit un message nerveux inhibiteur.

Vrai
Faux
Réponse : FAUX

Il n’y a pas de message nerveux inhibiteur qui commande au muscle de se relâcher. Le muscle se relâche simplement parce qu’il ne reçoit plus le message de se contracter.

2. Les réflexes sont des mouvements involontaires générés par le système nerveux autonome.

Vrai
Faux
Réponse : FAUX

Ils sont générés par le système nerveux somatique (muscles squelettiques).

3. Un muscle fléchisseur est toujours relâché.

Vrai
Faux
Réponse : FAUX

4. Les muscles squelettiques sont rattachés aux os par des ligaments.

Vrai
Faux
Réponse : FAUX

Ils sont rattachés aux os par les tendons. Les ligaments relient les os entre eux.

5. Le réflexe est présent dès la naissance.

Vrai
Faux
Réponse : VRAI

6. La voie nerveuse efférente est motrice.

Vrai
Faux
Réponse : VRAI

7. Les motoneurones passent par la racine ventrale du nerf rachidien.

Vrai
Faux
Réponse : VRAI

8. Dans le réflexe myotatique, le muscle qui se contracte est celui qui perçoit le stimulus.

Vrai
Faux
Réponse : VRAI

9. Dans les différents réflexes, les effecteurs sont toujours les muscles.

Vrai
Faux
Réponse : VRAI

11. Le récepteur sensoriel du réflexe myotatique est situé au niveau du tendon.

Vrai
Faux
Réponse : FAUX

Il se situe dans le corps même du muscle, en profondeur (fuseau neuromusculaire).

12. Le fuseau neuromusculaire est sensible à la tension du muscle.

Vrai
Faux
Réponse : FAUX

Il détecte la modification de l'étirement (longueur). La tension concerne l'organe tendineux de Golgi.

13. Le fuseau neuromusculaire est un mécanorécepteur.

Vrai
Faux
Réponse : VRAI

14. Il y a dégénérescence des fibres nerveuses non reliées au corps cellulaire.

Vrai
Faux
Réponse : VRAI

15. Le corps cellulaire des neurones sensitifs est situé dans la substance grise de la moelle épinière.

Vrai
Faux
Réponse : FAUX

Ils sont situés dans le ganglion spinal (racine dorsale).

16. Un motoneurone innerve une seule fibre musculaire.

Vrai
Faux
Réponse : FAUX

Un motoneurone innerve plusieurs fibres musculaires (unité motrice).

17. Toutes les fibres innervées par les ramifications du même motoneurone se contractent en même temps.

Vrai
Faux
Réponse : VRAI

18. Les neurones sensitifs sont amyéliniques et les motoneurones sont myélinisés.

Vrai
Faux
Réponse : FAUX

Les neurones sensitifs impliqués dans ces réflexes sont également myélinisés pour une conduction rapide.

19. Tous les réflexes médullaires sont monosynaptiques.

Vrai
Faux
Réponse : FAUX

Seul le réflexe myotatique est monosynaptique. Les autres (flexion, etc.) sont polysynaptiques.

20. Un circuit monosynaptique signifie que le neurone sensitif fait directement synapse avec le motoneurone.

Vrai
Faux
Réponse : VRAI

21. Pour qu’un muscle se contracte, il faut que le muscle antagoniste se relâche.

Vrai
Faux
Réponse : VRAI

22. Le relâchement du muscle antagoniste nécessite des messages nerveux inhibiteurs de la contraction.

Vrai
Faux
Réponse : FAUX

Comme pour la question 1, il n'y a pas de message inhibiteur direct au muscle. C'est l'absence de message excitateur qui induit le relâchement.

23. L’innervation réciproque des muscles antagonistes est un circuit monosynaptique.

Vrai
Faux
Réponse : FAUX

C'est un circuit polysynaptique car il nécessite au moins un interneurone inhibiteur.

24. Le neurone sensitif se connecte par une synapse inhibitrice au motoneurone du muscle antagoniste.

Vrai
Faux
Réponse : FAUX

Le neurone sensitif est toujours excitateur. Il stimule un interneurone, qui lui, possède une synapse inhibitrice avec le motoneurone antagoniste.

25. L’organe tendineux de Golgi est stimulé par une forte tension sur le tendon pendant un temps long uniquement.

Vrai
Faux
Réponse : FAUX

Il est stimulé par une forte tension, qu'elle soit courte ou longue.

26. La stimulation de l’organe tendineux de Golgi stimule le motoneurone du muscle contracté.

Vrai
Faux
Réponse : FAUX

Ce réflexe entraîne le relâchement du muscle, donc le motoneurone est inhibé (via un interneurone), pas stimulé.

27. On peut qualifier le réflexe tendineux de Golgi de "réflexe myotatique inverse".

Vrai
Faux
Réponse : VRAI

Il provoque le relâchement du muscle contracté pour le protéger.

28. Le réflexe ipsilatéral de flexion est un réflexe protecteur suite à une douleur.

Vrai
Faux
Réponse : VRAI

29. Le réflexe ipsilatéral de flexion fait intervenir des interneurones inhibiteurs.

Vrai
Faux
Réponse : FAUX

Ce sont des interneurones activateurs (excitants les fléchisseurs) qui interviennent.

30. Le réflexe ipsilatéral de flexion excite les extenseurs et inhibe les fléchisseurs controlatéraux.

Vrai
Faux
Réponse : FAUX

C’est le contraire pour permettre le retrait du membre stimulé.
