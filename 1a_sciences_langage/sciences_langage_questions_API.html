Exercices de Phonétique (API)

Exercice 1 : Description articulatoire complète

Donnez la description articulatoire complète des sons suivants (Mode, Lieu, Nasalité/Oralité, Voisement).

1- Décrivez le son [ŋ] (ou n vélaire) :

Réponse attendue :

Consonne occlusive, vélaire, nasale, voisée

2- Décrivez le son [ɔ] :

Réponse attendue :

Voyelle postérieure mi ouverte, orale, non étirée

3- Décrivez le son [ɥ] :

Réponse attendue :

Approximante (semi-voyelle / semi-consonne), labiopalatale, orale, voisée

4- Décrivez le son [v] :

Réponse attendue :

Consonne fricative, labiodentale, orale, voisée

5- Décrivez le son [p] :

Réponse attendue :

Consonne occlusive, bilabiale, orale, sourde (non voisée).

6- Décrivez le son [b] :

Réponse attendue :

Consonne occlusive, bilabiale, orale, voisée.

7- Décrivez le son [t] :

Réponse attendue :

Consonne occlusive, alvéolaire (ou dentale), orale, sourde.

8- Décrivez le son [d] :

Réponse attendue :

Consonne occlusive, alvéolaire (ou dentale), orale, voisée.

9- Décrivez le son [k] :

Réponse attendue :

Consonne occlusive, vélaire, orale, sourde.

10- Décrivez le son [g] :

Réponse attendue :

Consonne occlusive, vélaire, orale, voisée.

11- Décrivez le son [f] :

Réponse attendue :

Consonne fricative, labio-dentale, orale, sourde.

12- Décrivez le son [s] :

Réponse attendue :

Consonne fricative, alvéolaire, orale, sourde.

13- Décrivez le son [z] :

Réponse attendue :

Consonne fricative, alvéolaire, orale, voisée.

14- Décrivez le son [ʃ] (ch) :

Réponse attendue :

Consonne fricative, post-alvéolaire (ou palato-alvéolaire), orale, sourde.

15- Décrivez le son [ʒ] (j) :

Réponse attendue :

Consonne fricative, post-alvéolaire (ou palato-alvéolaire), orale, voisée.

16- Décrivez le son [l] :

Réponse attendue :

Consonne latérale (liquide), alvéolaire, orale, voisée.

17- Décrivez le son [R] (r français standard) :

Réponse attendue :

Consonne fricative (ou vibrante), uvulaire, orale, voisée.

18- Décrivez le son [i] :

Réponse attendue :

Voyelle fermée, antérieure, orale, non-arrondie (étirée).

19- Décrivez le son [u] (ou) :

Réponse attendue :

Voyelle fermée, postérieure, orale, arrondie.

20- Décrivez le son [a] :

Réponse attendue :

Voyelle ouverte, antérieure, orale, non-arrondie (étirée).

Exercice 2 : Transcription API depuis la définition

Donnez le signe API qui correspond aux définitions suivantes.

1- Consonne, occlusive, bilabiale, orale, voisée:

Réponse attendue :

[b]

2- Consonne, occlusive, palatale, nasale, voisée:

Réponse attendue :

[ɲ] (gn)

3- Voyelle, antérieure, mi-ouverte, orale, étirée:

Réponse attendue :

[ε]

4- Consonne, constrictive, uvulaire, orale, non voisée:

Réponse attendue :

[x] ou [χ]

5- Voyelle, postérieure, mi-fermée, orale, arrondie:

Réponse attendue :

[o]

6- Consonne, occlusive, bilabiale, nasale, voisée:

Réponse attendue :

[m]

7- Voyelle, antérieure, mi-ouverte, nasale, non-arrondie:

Réponse attendue :

[ɛ̃] (in)

8- Consonne, fricative, post-alvéolaire, orale, voisée:

Réponse attendue :

[ʒ] (j)

9- Consonne, occlusive, bilabiale, orale, sourde :

Réponse attendue :

[p]

10- Consonne, occlusive, dentale (ou alvéolaire), orale, sourde :

Réponse attendue :

[t]

11- Consonne, occlusive, dentale (ou alvéolaire), orale, voisée :

Réponse attendue :

[d]

12- Consonne, occlusive, vélaire, orale, sourde :

Réponse attendue :

[k]

13- Consonne, occlusive, vélaire, orale, voisée :

Réponse attendue :

[g]

14- Consonne, occlusive, dentale, nasale, voisée :

Réponse attendue :

[n]

15- Consonne, fricative, labio-dentale, orale, sourde :

Réponse attendue :

[f]

16- Consonne, fricative, alvéolaire, orale, sourde :

Réponse attendue :

[s]

17- Consonne, fricative, alvéolaire, orale, voisée :

Réponse attendue :

[z]

18- Consonne, fricative, post-alvéolaire, orale, sourde :

Réponse attendue :

[ʃ] (ch)

19- Consonne, latérale, alvéolaire, orale, voisée :

Réponse attendue :

[l]

20- Consonne, fricative/vibrante, uvulaire, orale, voisée :

Réponse attendue :

[R] ou [ʁ]

21- Voyelle, fermée, antérieure, orale, étirée :

Réponse attendue :

[i]

22- Voyelle, fermée, antérieure, orale, arrondie :

Réponse attendue :

[y] (u)

23- Voyelle, fermée, postérieure, orale, arrondie :

Réponse attendue :

[u] (ou)

24- Voyelle, mi-fermée, antérieure, orale, étirée :

Réponse attendue :

[e] (é)

25- Voyelle, mi-fermée, antérieure, orale, arrondie :

Réponse attendue :

[ø] (eu fermé, comme "feu")

26- Voyelle, centrale, moyenne, orale, légèrement arrondie (schwa) :

Réponse attendue :

[ə] (e muet, comme "le")

27- Voyelle, ouverte, antérieure, orale, étirée :

Réponse attendue :

[a] (patte)

28- Voyelle, ouverte, postérieure, nasale, arrondie :

Réponse attendue :

[ɑ̃] (an, en)

29- Voyelle, mi-ouverte, postérieure, nasale, arrondie :

Réponse attendue :

[ɔ̃] (on)

30- Semi-consonne, labio-vélaire, orale, voisée :

Réponse attendue :

[w] (oui)
